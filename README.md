# Personnality Site - Version Finale (Multilingue)

## Mise à jour (11 Janvier 2026) - Internationalisation & Précision

### Nouveautés Majeures
-   **Support Anglais & Français** : Le site est maintenant entièrement bilingue.
    -   Un bouton "FR/EN" dans le header permet de changer la langue à la volée.
    -   Toutes les 180+ questions et l'interface ont été traduites.
-   **Affichage des Pourcentages** : Les résultats MBTI affichent désormais le pourcentage précis pour chaque trait (ex: "78% Introverti").
-   **Lien vers Description Détaillée** :
    -   En **Français**, le résultat renvoie vers votre fichier local (`personnalite/mbti/fr/types-de-personnalite.html`).
    -   En **Anglais**, le résultat renvoie vers le site officiel 16Personalities (fallback).

### Fonctionnalités Précédentes
-   **Mode Sombre Avancé** : Toggle manuel et détection auto.
-   **PDF Export** : Génération de certificat avec graphiques.
-   **Zéro Budget** : Tout fonctionne en local sans API payante.

### Utilisation
Ouvrez `index.html`. 
-   Utilisez le bouton **FR/EN** pour changer la langue.
-   Utilisez la Lune/Soleil pour le thème.
